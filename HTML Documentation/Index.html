ldap¶

Version:3.4.x
Language:en
Authors:Norman Seibert
Email:seibert@entios.de
License:This extension documentation is published under the CC BY-NC-SA 4.0 (Creative Commons) license

ldap is an extension to connect TYPO3 to LDAP directories and to fetch user records from them.
